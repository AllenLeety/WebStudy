GitHub服务器端修改的内容
Git GitHub
1.GitHub上传一个文件：
（1）git init         初始化一个./git来管理本地git
（2）git add 文件名
（3）git commit 文件名 -m "文件详细描述"
（4）git remote add origin "github url"
（5）git push -u origin master
2.本地修改文件，然后再上传GitHub：
（1）git status
（2）git diff     查看不同
（3）git pull
（4）git add 文件名
（5）git commit 文件名 -m "文件详细描述"
（6）git push -u origin master
